# TalLabelRouter-label-router-releases

Automatically detect, extract, and print labels from your e-commerce PDFs. Built for sellers who handle multiple label types daily.

## What it does

TalLabelRouter watches your downloads folder for new PDF files and automatically:

- **Detects** the label type (product barcodes, shipping labels, courier stickers, booking confirmations)
- **Extracts** individual labels from multi-label sheets
- **Routes** each label type to the correct printer
- **Prints** directly to thermal and label printers without opening a PDF viewer

No more manually opening PDFs, cropping labels, and selecting printers. Drop a file and it's printed.

## System Requirements

- Windows 10 or Windows 11
- Internet connection (required for license activation)
- One or more printers configured

## Installation

1. Download `TalLabelRouter.exe` from the [Releases](https://github.com/ayvadia/TalLabelRouter-releases/releases) page
2. Run the exe — it installs automatically
3. Enter your license key when prompted
4. Configure your printers and watch folder
5. Start monitoring — labels print automatically

The app runs in your system tray and monitors your folder in the background.

## Features

- **6 label types** supported: product barcodes, shipping labels, Makro labels, Shiplogic stickers, Adaptris bookings, shipping notes
- **Per-type printer routing** — send each label type to a different printer
- **Direct printing** — prints to thermal printers without needing a PDF viewer
- **Auto-print mode** — new PDFs are detected, extracted, and printed automatically
- **Drag and drop** — drop a PDF onto the window for quick one-off processing
- **System tray** — runs silently in the background
- **Auto-updates** — the app checks for updates and installs them automatically
- **Windows startup** — optionally start with Windows
